Inky


Key Points
  • Built an iOS app using Swift that transcribes lecture audio files with Swift’s SF Speech Recognizer and summarized them using an external Natural Language Processing library
  • It has an option to record live audio or import pre-existing audio files when choosing files to summarize. All programming is done using Swift on Xcode.
  • Contributors: Enrique Moran, Ronin Cunningham, Ariel Baena-Tan, and Joshua Park
